Wat is de return value van de confirm functie als de gebruiker op een van de buttons klikt?

Je krijgt een true terug als je op ok klikt en false als je op annuleren klikt

Wat is de return value van de prompt functie als de gebruiker een tekst intypt en op 'OK' klikt? Je krijgt dan de waarde die je hebt meegegeven terug. (In mijn geval 'onbekend') Als je op annuleren klikt krijg je null terug.
